# PHP Exercises Repository

## 📌 Description  
This repository contains **two directories**, each with exercises corresponding to different levels.  
The exercises are designed to test **the fundamentals of PHP**, including:  
- Declaring variables  
- Using functions  
- Conditional statements  
- Loops    

## 🛠️ Technologies Used  
- **PHP 8.2**  
- **Visual Studio Code**  
- **XAMPP** (for local development)  
- **Git & GitHub** (for version control)
